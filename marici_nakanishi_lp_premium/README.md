## 🎭 和のディテール

- **章の一、章の二…**: 各セクションを「書物の章」として構造化
- **縦書きの引用句**: `writing-mode: vertical-rl` で伝統的な縦組みを再現
- **漢数字 01/02/03**: セリフ体の漢数字を章番号の装飾として配置
- **「艶」「潤」「彩」**: Craft 3 つに対応する一字の漢字を大きく配置
- **和装の角飾り**: `.tl-corner / .br-corner` で画像フレームに一桁の風合いを追加

## 🏗️ 技術的な仕事

### カスタム CSS アーキテクチャ

- **41 個の CSS カスタムプロパティ** でテーマ一元管理 (色・書体・イージング)
- **インライン CDN** ではなく **手書きの設計された CSS**
- モーション: `cubic-bezier(0.16, 1, 0.3, 1)` (自然な減速)
- `prefers-reduced-motion` 対応 (視覚過敏・WCAG 対応)

## 🎯 制作ハイライト

### 1. 時間軸を持つ物語構造
普通の美容室 HP は「自己紹介+メニュー+予約」の一枚もので終わるが、本ページは**読み物として成立する編集設計**。

### 2. Craft セクションの演出
各技術を「核となる三つ」として等価に並べ、それぞれに**漢字一文字のアイコン・二段階カプセル化・長文本文**で深掘り。

### 3. Course (Menu) のレストラン化
価格表を「コース仕立て」に。各メニューに `01, 02, 03...` の番号、`Signature / Popular / Hightone / Premium / Counsel` の英字ラベル。

### 4. 漢数字章見出し
各セクションの冒頭に **「一、二、三...」の縦書き漢数字**。章立てという身体感覚を読み手に与える。

### 5. Instagram との一体化
投稿が自動更新される埋め込みを、サロンのブランディングと矛盾しない枠で実装。
