## ファイル形式

- **JPG** 推奨 (品質 80–90% で書き出し)
- WebP でも OK だが、その場合 `index.html` 内の `.jpg` を `.webp` に置換してください
- ファイルサイズ目安: 1 枚 200KB〜800KB (合計 5MB 以下が理想)

## 圧縮ツールおすすめ

- [Squoosh](https://squoosh.app/) — ブラウザで使える Google 製
- [TinyJPG](https://tinyjpg.com/) — 一括投げ込みで OK

## 確認方法

ファイルを保存したら、`index.html` をダブルクリックでブラウザに開けば確認できます。
