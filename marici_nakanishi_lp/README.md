## 🚀 公開方法（3パターン）

### A. Vercel / Cloudflare Pages で無料公開（推奨）

1. GitHub にリポジトリ作成 → `index.html` をアップロード
2. Vercel / Cloudflare Pages と連携 → 1クリックで公開
3. 独自ドメイン（例: `nakanishi-marici.com`）を設定可

### B. お手元のサーバーに FTP アップロード

`index.html` を Web サーバーのルートにアップロードするだけで完了。

### C. 即確認用にローカルで開く

`index.html` をダブルクリック → ブラウザで即表示（Instagram 埋め込みはオンライン時のみ動作）。
